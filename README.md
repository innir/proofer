# Temperature control with timer for a homemade proofing box


## BOM

- LOLIN D1 mini (Pro)
- SSR RP1A23D5
- Power supply IRM-05-5
- Resistor 4.7 kOhm
- Terminal WAGO 250-202
- Terminal WAGO 250-203
- Housing GEH KSW 50 (G086)
- Temperature sensor DS18B20


- Heating cable / mat, 230V, ~20W
- Insulation box
